## 修改需求
### 需求：对评测结果执行智能评分
系统必须在执行智能评分时，仅使用标注结果、模型输出和评分标准作为评分输入，并继续由大模型返回 `recall`、`accuracy`、`reason`。

#### 场景：评分输入仅包含三项内容
- **当** 系统对单条评测结果执行智能评分
- **那么** 系统仅向评分模型提供 `ground_truth`、`model_output` 和 `评分标准`

#### 场景：不传入任务 Prompt 与自定义标签
- **当** 系统执行智能评分
- **那么** 系统不再将任务 `prompt` 和评测集 `custom_tags` 作为评分输入的一部分
