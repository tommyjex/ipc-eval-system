# 变更：添加评测集管理功能

## 为什么
评测集管理是评测平台的核心功能，用户需要创建评测集、上传评测数据、进行数据标注，为后续的模型评测提供数据基础。该功能支持视频和图片两种评测集类型，并提供灵活的数据导入和标注方式。

## 变更内容
- 创建评测集管理模块（前端界面 + 后端API）
- 支持视频类型和图片类型评测集的创建
- 支持主流视频格式（MP4、AVI、MOV等）和图片格式（JPG、PNG、GIF等）
- 支持本地上传和从TOS对象存储导入两种数据导入方式
- 本地上传的数据自动写入TOS，object key存储到MySQL
- 创建数据标注模块，支持大模型自动标注和人工标注
- 支持批量标注和逐条标注两种标注模式

## 影响
- 受影响规范：dataset-management（新建）、data-annotation（新建）
- 受影响代码：
  - 前端：新增评测集管理页面、数据上传组件、标注界面
  - 后端：新增评测集API、文件上传服务、TOS集成、标注服务
  - 数据库：新增评测集表、评测数据表、标注记录表
