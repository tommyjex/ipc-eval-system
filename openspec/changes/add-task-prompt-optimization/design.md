## 上下文
任务详情页当前已经有三类关键信息：
- 任务级 Prompt
- 全量结果明细（对象名称、标注结果、模型输出）
- 智能评分结果（召回率、精确率、评分理由）

这些信息足以支撑“基于全量样本 diff 的提示词诊断”。但当前系统没有任何聚合分析能力，用户只能逐条查看结果，无法快速定位“模型漏掉了什么”“模型稳定误报什么”“Prompt 是否对输出格式、关注对象、负例约束表达不足”。

## 目标
- 在任务详情页提供单次触发、可重复执行的提示词优化能力
- 使用已评分样本自动总结问题模式，并输出可直接试用的优化后提示词
- 支持用户对优化后提示词做人工微调
- 支持基于优化后提示词新建对比任务并自动完成评测与智能评分
- 支持从任务级指标维度对比优化前后的效果
- 支持用户将优化后提示词一键替换为当前任务 Prompt

## 非目标
- 不在本次变更中为每条样本单独生成优化建议
- 不引入 Prompt 版本管理、多人协作审批或历史 diff 对比
- 不扩展到任务列表页批量触发

## 决策
### 1. 入口限制在任务详情页
只在任务详情页提供“提示词优化”按钮，避免在任务列表页引入新的复杂状态管理。用户先查看评分结果，再决定是否触发优化。

### 2. 输入采用全量已评分样本
优化任务的输入应来自当前评测任务下的全量已评分样本，至少包括：
- 对象名称
- 标注结果
- 模型输出
- 召回率
- 精确率
- 评分理由

如果任务尚未完成智能评分，系统不应允许触发提示词优化。

### 3. 重点基于 diff 聚合问题模式
优化链路不只是“把所有样本拼给模型”，而应在服务端先整理每条样本的关键比较信息，突出：
- 漏检项
- 误报项
- 空样本误报
- 低召回/低精确率样本
- 高频失败类型

第一阶段模型重点输出：
- 问题点总结
- 根因归纳
- 优化策略

第二阶段模型重点输出：
- 完整优化后提示词
- 对关键修改点的简短说明

### 4. 固定优化模型
第一版固定使用 `doubao-seed-2-0-pro-260215`，并开启 thinking 模式，避免把“优化模型配置”也变成一个新配置面。

### 5. 两阶段优化链路
第一版采用两次串行模型调用：
- 第一阶段：基于全量已评分样本生成结构化问题分析与优化策略
- 第二阶段：基于原始 Prompt 和第一阶段输出生成优化后提示词

这样可以把“诊断”和“改写”分离，降低模型把分析和生成混在一起的风险，同时增强结果可解释性。

### 6. 优化结果可编辑
优化后提示词不应只是只读展示。用户需要在系统内直接修改优化后提示词，形成“模型建议 + 人工微调”的工作流。

### 7. 默认通过新建对比任务重跑
当用户确认使用优化后提示词重新验证效果时，系统默认不覆盖原任务，而是创建一个新的对比任务：
- 继承原任务的评测集、目标模型、模型供应商和其他必要参数
- 使用人工微调后的优化提示词作为新任务 Prompt
- 自动运行评测
- 自动触发智能评分

这样可以保留原任务基线，确保优化前后对比可追溯。

### 8. 提供任务级效果对比
第一版默认提供任务级指标对比，而不是逐样本比对。重点展示：
- micro recall / precision
- macro recall / precision
- coverage rate
- empty sample pass rate
- unscorable count

### 9. 支持显式替换当前任务 Prompt
在用户确认后，系统应支持把优化后提示词写回当前任务 Prompt。该动作必须显式触发，不能自动执行。

## 风险 / 权衡
- 全量样本直接拼接可能导致上下文过长，需要后端做摘要和抽样
- 优化建议受固定模型能力影响，输出可能存在泛化建议，需要保留人工采纳
- 若任务样本很多，优化接口需要异步化或至少给出处理中反馈
- 自动创建对比任务会引入新的任务状态与生命周期，需要避免和原任务混淆
- 如果允许替换当前任务 Prompt，需要明确“替换 Prompt”与“新建对比任务”是两个独立动作

## 建议实现方向
- 后端新增“提示词优化任务”接口，按任务维度聚合样本
- 先做服务端摘要：统计低分样本、抽取典型 diff、汇总问题模式
- 再调用固定优化模型生成第一阶段问题分析
- 最后将原始 Prompt + 第一阶段输出送入第二阶段模型生成优化后提示词
- 前端展示四块内容：
  - 总体问题摘要
  - 典型 diff/问题点
  - 优化策略
  - 优化后提示词
- 前端允许用户直接编辑优化后提示词
- 用户可触发“新建对比任务并自动运行评分”
- 系统在原任务和对比任务之间展示任务级指标对比
- 用户可显式点击“替换为当前任务 Prompt”将编辑后的提示词写回当前任务
