## 新增需求
### 需求：任务 Prompt 模板
系统必须支持任务 Prompt 模板的创建、编辑、删除与查询，模板可被任务创建和任务编辑时复用。

#### 场景：创建 Prompt 模板并按场景查询
- **当** 用户创建任务 Prompt 模板并指定业务场景
- **那么** 系统保存该模板，并允许按业务场景查询模板列表

#### 场景：创建任务时选择 Prompt 模板
- **当** 用户在创建任务时选择任务 Prompt 模板
- **那么** 系统将模板内容覆盖到任务的 `prompt` 输入框

#### 场景：编辑任务时选择 Prompt 模板
- **当** 用户在任务详情编辑中选择任务 Prompt 模板
- **那么** 系统将模板内容覆盖到任务的 `prompt` 输入框

### 需求：Prompt 模板业务场景分类复用评测集场景
Prompt 模板的业务场景字段必须复用评测集的场景分类枚举。

#### 场景：模板场景与评测集场景一致
- **当** 用户创建 Prompt 模板时选择业务场景
- **那么** 可选业务场景与评测集 `scene` 枚举一致
