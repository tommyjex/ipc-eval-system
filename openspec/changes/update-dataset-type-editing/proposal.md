# 变更：支持在评测集详情页修改评测集类型

## 为什么
当前评测集详情页支持编辑名称、业务场景、标注提示词和自定义标签，但不支持修改评测集类型。对于图片类型评测集，用户无法在详情页直接修正或调整类型配置，导致配置维护成本较高。

## 变更内容
- 在评测集详情页增加评测集类型编辑能力
- 支持修改为 `图片类型`，并与现有 `视频类型 / 混合类型` 一起统一处理
- 后端数据集更新接口支持更新 `type`

## 影响
- 受影响规范：dataset-management（修改）
- 受影响代码：
  - 前端：评测集详情页增加类型编辑与保存交互
  - 前端：数据集更新接口参数类型支持 `type`
  - 后端：`DatasetUpdate` schema 支持 `type`
  - 后端：数据集更新接口处理 `type` 字段
