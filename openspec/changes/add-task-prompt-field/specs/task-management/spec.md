## 修改需求
### 需求：在任务详情页编辑任务信息
系统必须允许用户在任务详情页修改任务名称、评分标准和任务级 Prompt，并在保存成功后展示最新任务信息。

#### 场景：编辑任务名称
- **当** 用户进入任务详情页并打开编辑任务操作
- **那么** 系统展示当前任务名称并允许用户修改后保存

#### 场景：编辑评分标准
- **当** 用户进入任务详情页并编辑评分标准
- **那么** 系统保存更新后的评分标准并在详情页显示最新内容

#### 场景：编辑任务级 Prompt
- **当** 用户进入任务详情页并编辑任务级 Prompt
- **那么** 系统保存更新后的 Prompt 并在详情页显示最新内容

#### 场景：保存后刷新详情
- **当** 用户在任务详情页保存任务名称、评分标准或 Prompt 修改成功
- **那么** 系统重新加载任务详情并展示更新后的字段值

## 新增需求
### 需求：创建评测任务时配置任务级 Prompt
系统必须允许用户在创建评测任务时填写可选 Prompt，并在任务运行时优先使用该 Prompt。

#### 场景：不填写 Prompt 创建任务
- **当** 用户创建评测任务时不填写 Prompt
- **那么** 系统仍允许成功创建任务

#### 场景：填写 Prompt 创建任务
- **当** 用户在创建评测任务时填写 Prompt
- **那么** 系统保存该 Prompt 并使任务详情页可查看该内容

#### 场景：任务运行优先使用任务级 Prompt
- **当** 任务存在已保存的任务级 Prompt 并发起运行
- **那么** 系统调用大模型时必须优先使用该 Prompt
