## 修改需求
### 需求：支持人工微调优化后提示词
系统必须允许用户在任务详情页对优化后提示词进行人工编辑，然后再决定后续动作。

#### 场景：人工微调作用于指定优化版本
- **当** 用户查看某一版提示词优化结果
- **那么** 系统必须允许用户编辑该版本的优化后提示词文本
- **并且** 后续保存、创建对比任务、替换任务 Prompt 等动作必须明确作用于当前版本

### 需求：支持对比优化前后的任务级效果
系统必须支持展示优化前后的任务级指标对比。

#### 场景：对比卡片显示提升或下降
- **当** 系统展示原任务与对比任务的任务级指标对比
- **那么** 系统必须为每个指标显示提升、下降或持平的视觉反馈
- **并且** 系统必须使用颜色和箭头让用户快速识别变化方向

## 新增需求
### 需求：支持提示词优化历史版本
系统必须保留同一任务下的多版提示词优化结果，而不是在重新生成时覆盖旧版本。

#### 场景：重新生成时创建新版本
- **当** 用户在已有优化结果的任务上再次执行提示词优化
- **那么** 系统必须创建一条新的优化版本记录
- **并且** 系统不得覆盖已有历史版本

#### 场景：查看历史版本
- **当** 任务存在多个提示词优化版本
- **那么** 系统必须允许用户查看版本列表
- **并且** 系统必须支持切换查看任一版本的分析结果、优化后提示词与对比状态
