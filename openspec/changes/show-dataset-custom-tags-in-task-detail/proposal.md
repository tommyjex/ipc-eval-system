# 变更：在任务详情页展示评测集自定义标签

## 为什么
智能消息告警场景的评测集通常依赖自定义标签描述事件维度。当前任务详情页虽然已展示评测集名称，但无法直接看到评测集的自定义标签，用户在查看任务配置和结果时缺少关键上下文。

## 变更内容
- 在评测任务详情页展示所属评测集的自定义标签
- 仅在评测集存在自定义标签时显示该区域
- 复用现有评测集详情接口获取 `custom_tags`

## 影响
- 受影响规范：task-management（修改）
- 受影响代码：
  - 前端：任务详情页增加评测集自定义标签展示区域
  - 前端：复用现有 `datasetApi.get()` 返回的 `custom_tags`
  - 后端：无新增接口或数据结构变更
